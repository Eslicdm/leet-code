# GCP Analytics

Analytics services cover stream and batch processing, data warehousing, and business intelligence.

### Decision Path

| Need | Start with |
| :-- | :-- |
| Process streaming and batch data with Apache Beam. | [Dataflow](./dataflow.md) |
| Run Spark or Hadoop workloads. | [Dataproc](./dataproc.md) |
| Build interactive dashboards and BI reports. | [Looker](./looker.md) |

### When to Use GCP Analytics

- You need unified batch and streaming processing with exactly-once semantics.
- You want managed Spark or Hadoop clusters.
- You need governed BI with a single semantic layer.
