# GCP Monitoring

Monitoring services cover metrics, logs, traces, and error reporting.

### Decision Path

| Need | Start with |
| :-- | :-- |
| Collect metrics, build dashboards, and alert on SLOs. | [Cloud Monitoring](./cloud-monitoring.md) |

### When to Use GCP Monitoring

- You want metrics, logs, and traces in a single pane of glass.
- You need alerting on SLOs, budgets, or custom metrics.
- You want managed integration with other GCP services.
