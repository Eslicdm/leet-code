# Azure Analytics

Analytics services cover data warehousing, ETL, and big data processing.

### Decision Path

| Need | Start with |
| :-- | :-- |
| Run a managed data warehouse or query the data lake. | [Synapse Analytics](./synapse-analytics.md) |
| Build ETL pipelines across cloud and on-premises. | [Data Factory](./data-factory.md) |

### When to Use Azure Analytics

- You need a managed data warehouse with MPP.
- You need to query Data Lake Storage with SQL or Spark.
- You need ETL pipelines with 100+ connectors.
