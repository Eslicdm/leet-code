# Azure Serverless

Serverless services cover workflow automation, orchestration, and event-driven compute without managing servers.

### Decision Path

| Need | Start with |
| :-- | :-- |
| Automate workflows across SaaS and enterprise systems. | [Logic Apps](./logic-apps.md) |
| Write stateful workflows in code. | [Durable Functions](./durable-functions.md) |

### When to Use Azure Serverless

- You need a visual, no-code workflow builder with 400+ connectors.
- You need reliable, stateful orchestration in code.
- You want to reduce operational overhead for workflow automation.
