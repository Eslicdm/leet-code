# AWS DevOps

DevOps services cover infrastructure as code, CI/CD, and deployment automation.

### Decision Path

| Need | Start with |
| :-- | :-- |
| Define infrastructure as code in JSON or YAML. | [CloudFormation](./cloudformation.md) |
| Define infrastructure as code in a familiar programming language. | [CDK](./cdk.md) |

### When to Use AWS DevOps

- You want versioned, repeatable infrastructure.
- You need to integrate tightly with other AWS services.
- You want to use the same language your team already knows for IaC.
