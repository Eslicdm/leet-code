# AWS Analytics

Analytics services cover querying, streaming, and ETL over data at scale.

### Decision Path

| Need | Start with |
| :-- | :-- |
| Run ad hoc SQL queries over data in S3. | [Athena](./athena.md) |
| Ingest and process real-time streaming data. | [Kinesis](./kinesis.md) |
| Build and run ETL pipelines with a data catalog. | [Glue](./glue.md) |

### When to Use AWS Analytics

- You need interactive SQL queries over S3-based data lakes.
- You need real-time ingestion and processing of streaming data.
- You need managed ETL pipelines with a central data catalog.
